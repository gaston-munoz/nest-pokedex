## Stack

- NodeJS
- NestJS
- Typescript
- MongoDB
- Docker

## Installation

- Download the proyect
- Install dependencies running
```
$ yarn install
```

- Create __.env__ file with __.env.template__ model

## Running the app
- Up DB
```
$ docker-compose up -d
```

- Start the server
```
# development
$ yarn dev


# production mode
$ yarn start:prod
```
